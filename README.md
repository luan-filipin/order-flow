OrderFlow

Sistema de gerenciamento de entregas baseado em microsserviços e arquitetura orientada a eventos.

Tecnologias:
- Java 21
- Spring Boot
- Apache Kafka
- PostgreSQL
- Docker
- Spring Data JPA
- JUnit 5
- Mockito
- DBUnit

Objetivos:
- Demonstrar comunicação assíncrona entre serviços
- Aplicar conceitos de Event Driven Architecture
- Utilizar Kafka para processamento de pedidos em tempo real
- Implementar boas práticas de microsserviços
